Navbars

First use the tag nav to define the navbar and use the tag navel to define your navbar sub-items.

All navbar tags and sub-tags:

  • nav: Defines the navbar itself
  • navel: Defines the navbar element (like a link to another page)
  • navelfind: Adapts the style so an input field could fit in it.
  • navel + nav_disb (as class): Defines a disabled link in the menu
